### What this is

A one-page portfolio for my **self-directed, five-app "Accounting Meets AI" project** — AI-powered tools built from real accounting pain points, two deployed and live today.

### The portfolio

| App | What it does | Status |
|-----|--------------|--------|
| **PREPARE** | Reconciliation prep engine — statement PDFs → review-ready 1099 pre-reconciliation workbooks | Deployed · Live |
| **CASSIA** | Chat-based accounting support workspace (RAG + Text-to-SQL + recall) | Deployed · Live |
| **LUCENT** | Pre-audit review packet for general ledgers (ML + audit logic + validated memos) | Built · Deploying |
| **IRS Form Processor** | Guided 1099/W-9 preparation & review workspace | In design |
| **Tax Schedule Classifier** | Vendor/GL → Schedule C/E/F line-item candidates | In design |

All five converge into **TAU**, a single hub — where the portfolio finishes.
